# React boilerplate

Initial setup for React with Typescript projects.

## Installation

Clone the repository, then run:

```bash
npm install
```

## Usage

Development mode

```bash
npm start
```

Generate build

```bash
npm run build
```

Serve build

```bash
npx webpack serve
```
